vimrc
=====

My vim settings

Install
-------


```bash
git clone https://github.com/d4ndo/vimrc.git
cd vimrc
cp -R *.* ~/
```

Install Plugins
---------------


```bash
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

Start vim

and call :PlugInstall
